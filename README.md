#不务正业系列

> 不知道哪根筋抽了，突然想做这么个东西，最后竟然真做出来了。。
> 但是，没什么卵用啊！

代码虽然挺少，但该有功能的都有：
- mid文件解析
- mid音符代码、音符名称、音符实际频率的互相转换
- 使用 `Beep()` 进行播放（太难听别尝试。。）
- 模拟键盘操作，在[FreePiano]上弹奏
- 模拟鼠标操作，在[FreePiano]上弹奏（效果棒！）

mid文件的解析参考了开源项目[MIDI.js]，感谢~

[FreePiano]: http://freepiano.tiwb.com/
[MIDI.js]: https://github.com/mudcube/MIDI.js/
